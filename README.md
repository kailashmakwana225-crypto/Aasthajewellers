Aastha Jewellers - Single Page React (minimal)
---------------------------------------------
This ZIP contains a minimal Create React App structure with the AasthaJewellersLanding component.
Steps to run locally:

1. Unzip:
   unzip aastha-jewellers.zip
2. Install dependencies:
   npm install
3. Run locally:
   npm start
4. To build for production:
   npm run build
5. To deploy:
   - Recommended: Use Vercel (https://vercel.com). Sign up, connect GitHub, import repo and deploy.
   - Or use Netlify (https://app.netlify.com) or GitHub Pages (see instructions in the docs).

Helpful links (quick):
- Vercel deploy guide: https://vercel.com/docs/frameworks/frontend/create-react-app
- Netlify React guide: https://docs.netlify.com/build/frameworks/framework-setup-guides/react/
- GitHub push guide: https://docs.github.com/en/repositories/creating-and-managing-repositories/quickstart-for-repositories
